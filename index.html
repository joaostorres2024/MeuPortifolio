Sobre Mim!

Olá, meu nome é João e sou um desenvolvedor Front-End apaixonado por tecnologia. Busco constantemente aprender novas tecnologias por conta própria e o meu objetivo como desenvolvedor é criar interfaces modernas e de alta qualidade, com foco em performance e responsividade. Algumas das minhas habilidades como desenvolvedor Front-End incluem HTML, CSS, JavaScript e VueJS. Além disso, também gosto de estudar tecnologias de Back-End nas minhas horas vagas.

Meus Projetos

  • O redesign da interface da YourCLub.io, uma rede social inovadora, utilizou técnicas avançadas de UX para tornar a plataforma mais intuitiva e funcional. Com base em pesquisas de usuários e testes de usabilidade, foi desenvolvido um design clean, minimalista e inclusivo, focado na navegação fácil e acessível. Após o lançamento, a análise contínua de feedback garantiu melhorias contínuas. Ferramentas como Adobe XD, Figma, User Testing e Analytics foram essenciais no processo. Este projeto visa aumentar a satisfação, promover engajamento e melhorar a acessibilidade da plataforma.

  • O redesign da interface web da CONECTE PUBLI utilizou técnicas avançadas de UX para aprimorar a navegação e a usabilidade da plataforma. Foram realizadas pesquisas de usuários e testes de usabilidade para compreender suas necessidades e padrões de comportamento. A nova interface foi projetada com um layout limpo e minimalista, facilitando a navegação intuitiva e acessível para todos os usuários. Ferramentas como Adobe XD, Figma, User Testing e Analytics foram utilizadas no desenvolvimento e na análise contínua de feedback para garantir melhorias constantes. Este projeto visa aumentar a satisfação do usuário, promover maior engajamento e melhorar a acessibilidade da plataforma, tornando-a mais eficiente e agradável de usar.

  • A criação da interface web da StyleFitness, um site de academia em desenvolvimento, utiliza técnicas avançadas de UX para proporcionar uma experiência de navegação fluida e intuitiva. Desde o início, foram realizadas pesquisas de usuários e testes de usabilidade para entender suas necessidades e preferências, resultando em um layout clean e altamente funcional. A combinação de várias linguagens de programação, como HTML, CSS, JavaScript e PHP, permitiu desenvolver uma interface dinâmica e responsiva. Ferramentas como Adobe XD e Figma foram essenciais na elaboração do design, enquanto User Testing e Analytics ajudaram a ajustar e otimizar a experiência do usuário. Este projeto visa criar uma plataforma que aumente a satisfação do usuário, promova maior engajamento e garanta uma utilização eficiente por todos.

  • O projeto de design acadêmico do jogo Quiz Persona Quest focou na criação de uma interface acessível e envolvente para os jogadores. Aplicando técnicas de UX, foi conduzida uma série de pesquisas e testes de usabilidade para entender as expectativas e preferências dos usuários. O design desenvolvido desde o início utilizou linguagens de programação como HTML, CSS e JavaScript para criar uma interface interativa e responsiva. Com o auxílio de ferramentas como Adobe XD e Figma, foi possível elaborar um layout atraente e intuitivo, proporcionando uma experiência de jogo fluida. Este projeto, sendo meu primeiro design acadêmico, resultou em uma plataforma que busca aumentar o engajamento dos jogadores e proporcionar uma experiência de uso agradável e eficiente.
